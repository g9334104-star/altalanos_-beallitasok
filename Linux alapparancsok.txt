CHMOD – Jogosultságok
Számok jelentése
SzámJogosultságJelentés
0---semmi
1--Xcsak futtatás
2-W-csak írás
3-WXírás + futtatás
4R--csak olvasás
5R-Xolvasás + futtatás
6RW-olvasás + írás
7RWXminden jog
Struktúra
chmod XYZ fájl
      │││
      ││└── mások (others)
      │└─── csoport (group)
      └──── tulajdonos (owner)
Leggyakoribb chmod parancsok
bash
chmod 777 fájl    # mindenki mindent tehet (RWX RWX RWX)
chmod 755 fájl    # tulajdonos: RWX, csoport: R-X, mások: R-X
chmod 644 fájl    # tulajdonos: RW-, csoport: R--, mások: R--
chmod 600 fájl    # tulajdonos: RW-, csoport: ---, mások: ---
chmod 700 fájl    # tulajdonos: RWX, csoport: ---, mások: ---
chmod 664 fájl    # tulajdonos: RW-, csoport: RW-, mások: R--
chmod 444 fájl    # mindenki csak olvashat
chmod 000 fájl    # senki sem férhet hozzá
Rekurzív (mappára és tartalmára)
bas
hchmod -R 755 /mappa   # minden fájlra és almappára alkalmazza
Tulajdonos beállítása
bash
chown user fájl              # tulajdonos megváltoztatása
chown user:csoport fájl      # tulajdonos és csoport
chown -R user:csoport /mappa # rekurzív
